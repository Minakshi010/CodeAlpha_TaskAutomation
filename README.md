# 🌐 Webpage Title Scraper (GUI)

A Python GUI application that extracts the title of a webpage and saves it to a file.

## 🚀 Features

- Enter website URL
- Fetch webpage title
- Display title in GUI
- Save title to TXT file

## 🛠 Technologies Used

- Python
- Tkinter
- Requests
- BeautifulSoup
- File Handling

## 🧠 Concepts Used

- HTTP requests
- Web scraping
- GUI programming
- File handling
